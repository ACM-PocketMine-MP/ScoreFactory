# A simple virion for making Scoreboards in Minecraft Bedrock Edition.

### Poggit Setup

Edit the `.poggit.yml` in your repository and set it up like shown below.

```yml
--- 
branches:
- master
projects:
  PLUGIN_NAME:
    libs:
      - src: Ifera/ScoreFactory/ScoreFactory
        version: ^3.1.0
...
```
